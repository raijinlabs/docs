## Methods

### `Agents`

```typescript
import { RaijinLabsLucidAi } from "raijin-labs-lucid-ai";

const raijinLabsLucidAi = new RaijinLabsLucidAi();

async function run() {
  const result = await raijinLabsLucidAi.agents.initAgent({
    agentId: "<id>",
  });

  console.log(result);
}

run();
```
